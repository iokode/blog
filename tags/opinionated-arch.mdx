[OpinionatedArch](https://iokode.blog/posts/opinionated-arch) is an Arch-based distribution for one physical person who wants multiple work contexts without maintaining different system configurations for each user account. It combines separate login users, shared versioned dotfiles, automatic snapshots, and explicit recovery tools into one opinionated operating model.

It is part of the IOKode project and it is created by Ivan Montilla.
